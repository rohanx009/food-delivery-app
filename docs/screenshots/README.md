## How to Capture Screenshots

1. **Desktop Screenshots:**
   - Use Windows Snipping Tool (Windows Key + Shift + S)
   - Or use browser developer tools (F12) and take device screenshots

2. **Mobile Screenshots:**
   - Use browser responsive mode (F12 â†’ Device toolbar)
   - Set device to iPhone 12 Pro or similar
   - Capture screenshots

3. **Recommended Dimensions:**
   - Desktop: 1920x1080 (Full HD)
   - Mobile: 375x812 (iPhone X/11/12)
   - Tablet: 768x1024 (iPad)

4. **File Naming Convention:**
   - Use lowercase with hyphens
   - Be descriptive: `customer-dashboard-active-orders.png`
   - Add device prefix for responsive: `mobile-restaurant-list.png`
